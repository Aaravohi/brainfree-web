BrainFree Language Guide

BrainFree is a simplified language inspired by Brainf***, created by Aaravohi for fun and educational purposes. You manipulate memory cells with explicit commands to create output.

Cells: [cell_id:value:function]

  • cell_id: moves the pointer relative to current cell.
    • cell0 → start at cell 0
    • cell+1 → move one cell right
    • cell-2 → move two cells left
  • value: change numeric value.
    • +5 → add 5
    • -3 → subtract 3
    • → no change
  • function: input/output actions.
    • out → output ASCII char (.)
    • in → input a char (,)
    • → no function

Loops: { ... } repeats enclosed commands while current cell is nonzero (Brainf*** style [ ... ]).

Example:

[cell+2:+5:out]  → Move two cells forward, add 5, output character

Combine multiple cells and loops to create complex programs or print messages creatively.
